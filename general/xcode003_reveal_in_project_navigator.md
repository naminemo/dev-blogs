# 找出目前正在編輯的檔案

快速找出目前正在編輯的檔案在專案導覽列 (Project Navigator) 中的位置，有兩種主要且非常方便的方法。

## 方法一：使用 "Reveal in Project Navigator" 功能 (最常用)

這是最直接且最常用的方法。

在您正在編輯的檔案視窗中，點擊 Xcode 視窗頂部、導覽列右側的「小箭頭」圖示，或在檔案路徑上點擊右鍵。

在彈出的選單中，選擇 "Reveal in Project Navigator" (在專案導覽列中顯示)。

當您點擊這個選項後，Xcode 會自動在導覽列中展開對應的資料夾，並高亮顯示您正在編輯的檔案。

## 方法二：使用鍵盤快速鍵

如果您喜歡使用鍵盤，這個方法會更有效率。

按下 Shift + Command + J (Shift + ⌘ + J)。

這個快速鍵的作用與「Reveal in Project Navigator」完全相同，它會立即在導覽列中跳轉並高亮顯示您當前正在編輯的檔案。


#  iOS 模擬器中，開啟鍵盤

當你的 App 在模擬器中執行，並且你有一個 TextField 或 TextEditor 元件時，點擊該元件，如果虛擬鍵盤沒有自動出現，你可以使用以下快捷鍵：

⌘K (Command + K)
